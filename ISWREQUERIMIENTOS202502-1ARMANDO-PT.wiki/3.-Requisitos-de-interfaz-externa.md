## 3.1 Interfaces de Usuario

El sistema contará con una interfaz gráfica diseñada para que los estudiantes puedan inscribirse en cursos, revisar horarios y verificar cupos disponibles. Cada pantalla tendrá un diseño intuitivo, responsivo y accesible. Esto permitirá que TÚ uses la plataforma sin problemas desde el celular o desde el computador.


### Interfaz de la aplicación móvil

Lo que ve el estudiante incluye un menú simple con acceso a inscripción, horarios, pagos y notificaciones. La vista muestra la lista de cursos con su cupo actualizado y un botón de inscripción rápida. El diseño usa botones claros y textos cortos que facilitan la navegación.

## 3.3 Interfaces de software

El sistema de inscripción de cursos universitarios se conectará con diferentes aplicaciones para garantizar su correcto funcionamiento:


### API de pagos en línea


* Se integra con servicios como PayU o MercadoPago para procesar pagos de cursos de extensión. Permite confirmar transacciones y actualizar automáticamente el estado del estudiante.


### Servicio de correo electrónico


* Envía confirmaciones de inscripción, avisos de horarios y notificaciones importantes usando un servidor SMTP o una API como SendGrid.


### Sistema académico interno

* Se conecta por medio de servicios web para consultar cursos, horarios, cupos, profesores y registrar las inscripciones realizadas en la plataforma.


### Base de datos central

* Maneja la información de usuarios, cursos, inscripciones, pagos y roles con acceso seguro y controlado.


### 3.4 Interfaces de Comunicación

### HTTP/HTTPS

* Permite la comunicación segura entre el usuario y el sistema para realizar inscripciones, consultas y pagos.

### Servicio de correo electrónico @

* Envía confirmaciones de inscripción, recordatorios y notificaciones importantes a los estudiantes.

### Actualización en tiempo real

* La plataforma mostrará cambios de cupos y horarios sin necesidad de recargar la página, usando servicios de actualización automática.

### Conexión con el servidor académico

* La información de cursos, profesores y horarios se sincroniza mediante servicios web internos de la universidad.
